## 🚀 快速启动指令

```bash
# 完整的项目设置
git clone https://github.com/juzhiqiang/web3-school-backend.git
cd web3-school-backend
npm install

# 本地开发环境
ganache-cli --deterministic --accounts 10 &
npm run deploy:local
npm run test
npm run interact

# 检查项目状态
npm run size-check
npm run estimate-gas
npm run lint
```

## 💡 使用建议

1. **本地开发测试**

2. **测试网部署**
